## Utility Functions

Following are the utility functions used for our graphs.

### Utility Function 1

U(t, d) = log(t) - delta * log(d)

### Utility Function 2

U(t, d) = t^a - b * t * d(d)/dt

### Utility Function 3

U(t, d) = t(1 - pRTT)
